# Reading Trainer — 아이패드 앱 (Expo)

같은 Docker/Node 서버에 연결해 **PIN 로그인 + 친구 랭킹 + 읽기**를 아이패드에서 쓰는 Expo 앱.

## 준비
1. **서버 실행** (PC):
   ```
   cd ../server
   npm install
   npm start            # 또는 docker compose up -d
   ```
   PC의 LAN IP 확인(예: `192.168.1.200`), 방화벽 4000 포트 허용.

2. **앱 설치/실행** (PC):
   ```
   cd mobile
   npm install
   npx expo start
   ```
   - 버전 정렬이 필요하면: `npx expo install --fix`

3. **아이패드**: App Store에서 **Expo Go** 설치 → PC 터미널의 **QR 코드**를 카메라/Expo Go로 스캔.
   - PC와 아이패드가 **같은 와이파이**에 있어야 함.

## 사용
1. 첫 화면에서 **⚙ 서버** → 주소를 `http://<PC-IP>:4000` 로 설정(기본값은 `192.168.1.200:4000`).
2. 사용자 선택 → **4자리 PIN** 로그인(없으면 ＋ 새 사용자로 등록).
3. 홈: 내 기록 + **🏆 친구들과 비교**(연속일/이번주/글자/완독) + 저장된 글.
4. 글을 눌러 읽으면 세션이 서버에 올라가 랭킹에 반영.

## 메모
- 서버 API는 PC 앱과 동일(`/api/...`). 데이터(사용자·글·세션·랭킹)는 서버에 모입니다.
- 기기 실행 검증은 실제 아이패드 + Expo Go에서 진행하세요(이 저장소에선 코드만 제공).
