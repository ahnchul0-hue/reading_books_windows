# v2 개편 계획 — 사용자 대시보드 + UX 개선

브랜치: `feat/v2-dashboard`. 휴먼 확정 결정 반영.

## 확정 결정
1. 연속으로 = 멈춘 위치(쪽·줄)부터 재개
2. 꾸준함(streak) = 하루 1회 이상 읽으면 그날 인정
3. 시간 남으면 = 미리 고른 다음 글로 진행 (D4 "반복" 폐기)
4. 대시보드 = 연속일 · 최근기록 · **월별 캘린더(한 날/안 한 날 명확)** · 격려 + 저장글 아이콘 그리드
5. 반응형 = 글자 크기 유지, 줄바꿈/배치만 창에 맞춤
6. 기본값: 줄수 4 / 속도 1.0 / 시간 10분 / 테마 어둡게(기본) / 글자크기 +/-
